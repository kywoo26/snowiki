# Archive Structure

This directory holds superseded but still useful documentation that should remain available for lineage and traceability.

## What belongs here

- former root-level strategy/design documents
- historical design syntheses
- superseded documents that are still worth preserving

## What does not belong here

- current planning truth
- active architecture contracts
- current research inputs still used directly by the roadmap

For active planning, start with `docs/roadmap/`.
